# Tap FIFA — Pricing Model

## Packaging Logic
Pricing follows activation maturity:
- Base deployment fee (event setup + launch)
- Campaign fee (sponsor and reward operations)
- Performance upside (optional success-based component)

## Tier 1 — Launch
Best for: first activation, single venue pilot.

Includes:
- 5 Tap poster routes
- Branded Tap UI deployment
- Basic sponsor card integration
- Standard post-event report

Commercial structure:
- Platform fee: fixed
- Sponsor campaign fee: fixed per campaign
- Optional support add-on: daily rate

## Tier 2 — Growth
Best for: multi-event windows or larger sponsor programs.

Includes everything in Tier 1, plus:
- Multi-language sponsor targeting
- Reward challenge and redemption logic
- Enhanced analytics by route, language, and intent
- Weekly optimization cycle

Commercial structure:
- Platform fee: fixed monthly/event-window
- Campaign fee: fixed + volume discount
- Optional performance bonus tied to CTR/redemption uplift

## Tier 3 — Host-City Program
Best for: city-wide tournament operations.

Includes everything in Tier 2, plus:
- Multi-venue orchestration
- City-service and hospitality route expansion
- Executive reporting and sponsor portfolio roll-up
- Dedicated launch and optimization team

Commercial structure:
- Program fee: fixed per host-city window
- Sponsor portfolio fee: bundle pricing
- Performance component: negotiated against conversion KPIs

## KPI Anchors For Commercials
Suggested KPI families:
- Activation: scan volume, unique scanners, open rate
- Utility: map interactions, next-step taps, route completion actions
- Monetization: sponsor impression-to-click, redemption rate, attributable conversions
- Support: help-flow completion, fallback call/SMS usage

## Contract Shapes
Choose one based on buyer preference:
1. Fixed Fee: highest budget certainty
2. Hybrid: lower base + performance bonus
3. Sponsor-Funded: operator base offset by sponsor package commitments

## Upsell Path
1. Pilot event closes with KPI proof
2. Expand to additional zones/routes
3. Add rewards and premium sponsor packages
4. Convert to city-scale program
