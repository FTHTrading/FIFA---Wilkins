    Activation

    One command to prep the Atlanta FIFA demo

    pnpm activate:fth
    • Applies the FTHTrading brand pack to the GitHub Pages demo.
    • Copies `.env.example` into `.env` if needed.
    • Starts PostgreSQL and Redis via Docker Compose.
    • Generates Prisma, migrates the database, and seeds Atlanta data.
    • Launches the monorepo dev stack and opens the key URLs.
