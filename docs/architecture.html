Flow 1

Guest Web Concierge

A fan opens the web app, selects their language, asks a question, and receives a culturally-aware, sponsor-eligible response.

🌐
Fan opens Guest Web
Language selection screen. 10 languages supported. RTL detection for Arabic.
→
💬
Types question
"Where can I find halal food near the stadium?"
→
🧠
Intent Classification
API detects language (ar), classifies intent (food_search), extracts entities (halal, stadium).
→
🔍
RAG Retrieval
pgvector similarity search across venue POIs, restaurants, and city services. Cultural ranking applied.
→
💰
Sponsor Injection
Eligible sponsor offers (halal restaurant promos) injected based on campaign tier and cultural match.
→
✅
Response Delivered
Culturally-aware response in Arabic with restaurant recs, directions, and sponsor offers.
Flow 2

SMS / Voice Pipeline

A fan texts or calls the event number. Telnyx routes to the API, which detects language, classifies intent, and sends a response — all without downloading an app.

📱
Fan sends SMS
"¿Dónde puedo ver el partido?" → Telnyx number
→
📡
Telnyx Webhook
Inbound message hits API at /api/telecom/webhook. Session created, message logged.
→
🌍
Language Detection
Detects Spanish (es). Creates TelecomSession with language binding.
→
🧠
Intent → Concierge
Same agentic concierge pipeline: intent classification, RAG retrieval, cultural ranking.
→
💰
Sponsor Check
Eligible Spanish-language sponsor offers attached to response if campaign active.
→
📤
Outbound SMS
Response sent via Telnyx outbound API in Spanish. DeliveryStatus tracked.
Flow 3

Emergency Escalation

Safety always wins. When a fan triggers an emergency phrase, the system bypasses all AI and sponsor logic and delivers deterministic, pre-approved safety responses.

🆘
Emergency Detected
Fan says "I need a doctor" or triggers any of 10 safety phrases in any language.
→
🛑
AI Bypassed
No RAG, no sponsor injection, no hallucination. Deterministic path only.
→
📋
Approved Phrase
Pre-approved translation delivered in the fan's language. No AI generation.
→
🚨
Incident Created
EmergencyIncident + EscalationEvent logged. Staff console alerted immediately.
→
👤
Staff Responds
Staff console shows incident. Translation-assisted rapid reply. Human in the loop.

10 Pre-Approved Emergency Phrases

🏥 Medical Emergency 🔥 Fire 👮 Police/Security 👶 Lost Child 🚑 Ambulance ⚡ Evacuation 🆘 General Help 💊 Pharmacy/First Aid 🚻 Accessibility 🔒 Shelter in Place
Flow 4

Sponsor Injection Pipeline

Revenue doesn't interrupt the experience — it enhances it. Sponsors are matched by language, culture, intent, and location, then injected transparently.

🎯
Intent Signal
Fan intent: food_search, transport, hospitality, nightlife, shopping...
→
🏷️
Campaign Match
Active SponsorCampaigns filtered by language, locale, intent category, and cultural profile.
→
🏅
Tier Selection
Smart Placement (passive), Cultural Concierge (contextual), or Reward Engine (interactive).
→
📊
Attribution
Impression, click, conversion tracked per campaign. Revenue attributed to language + intent.
Tier 1

Smart Placement

Passive sponsor placements in concierge responses, map cards, and search results. Lowest touch, broadest reach.

Tier 2

Cultural Concierge

Contextual sponsor recommendations matched to team cultural profiles. Restaurant partnerships, cultural venues, language-specific offers.

Tier 3

Reward Engine

Interactive challenges, QR scavenger hunts, match-day passport stamps. Highest engagement, premium pricing.
