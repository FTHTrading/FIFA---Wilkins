# Branding Packs

This directory holds white-label brand packs for FIFA---Wilkins.

Usage:

1. Add a new JSON pack in `branding/brands/`.
2. Run `pnpm activate` or `pnpm activate:fth`.
3. The launcher copies the chosen pack into `docs/brand.json` so the GitHub Pages demo and repo-facing collateral reflect that client instantly.

Current packs:

- `default.json` — neutral Wilkins base
- `fthtrading-fifa-wilkins.json` — FTHTrading-owned Atlanta FIFA demo pack
